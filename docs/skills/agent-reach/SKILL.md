# Agent Reach — 联网搜索

让 AI Agent 具备搜索互联网的能力，获取最新信息。

## 使用方法

```bash
openclaw skill install Panniantong/agent-reach
```

## 功能

- 搜索网页内容
- 回答关于最新事件和新闻的问题
- 查找实时数据和文档
