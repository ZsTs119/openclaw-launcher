# Awesome OpenClaw Skills

社区维护的 OpenClaw 技能合集，涵盖多种场景。

## 浏览地址

https://github.com/VoltAgent/awesome-openclaw-skills

## 使用方式

浏览合集页面，找到需要的技能后使用以下命令安装：

```bash
openclaw skill install <author>/<skill-name>
```

## 涵盖分类

- 开发工具
- 搜索与信息获取
- 自动化任务
- 代码审查
- 文档生成
- 数据处理
